# BakAnime

BakAnime es una página web estática que muestra estrenos de anime, un ranking de series y un formulario básico de contacto. El proyecto está creado únicamente con **HTML y CSS**, con el propósito de practicar la maquetación y la organización de contenido web.

## Características del proyecto
- Barra de navegación con enlaces a secciones internas.
- Sección principal con tarjetas de estrenos de anime (imagen, título y botón).
- Lista de ranking con animes populares.
- Formulario de contacto con campos para comentario y correo electrónico.
- Enlaces de redes sociales.
- Diseño organizado utilizando clases y contenedores.

## Cómo desplegar el proyecto
1. Clona o descarga el repositorio.
    https://github.com/Said-Meza/proyecto_final.git
2. Verifica que los archivos `index.html` y `style.css` estén en la misma carpeta.
3. Abre el archivo **index.html** en cualquier navegador web.
4. No requiere instalación adicional ni servidor local.
